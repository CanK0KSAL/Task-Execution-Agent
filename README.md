# Task Execution Agent

Take-home project scaffold. Setup and usage will be documented in a later phase.

## Setup (placeholder)

```bash
uv sync
uv run python main.py
```

## Run the interactive CLI

```bash
uv sync
uv run python main.py
```

Use `/help` for commands (`/debug`, `/reset`, `/exit`). Example prompts also work with `300 EUR` instead of `€300` on narrow Windows consoles. Dev tools and tests: `uv sync --extra dev` then `uv run pytest`.
